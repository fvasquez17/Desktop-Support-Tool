Executive Summary

The Desktop Support Tool is a locally-executed Windows utility that processes zero Protected Health Information (PHI), makes zero external API calls, stores zero credentials, and operates under a least-privilege model with per-action UAC elevation.

Security Posture at a Glance

Log Retention Policy (AU-11)

  • Default retention: 90 days (configurable via LoggingService.RetentionDays)
  • Enforcement: Automated on application startup — files older than the retention period are deleted
  • Windows Event Log: Follows system Event Log retention policy (configured via Group Policy)
  • Recommendation: Set Event Log maximum size to 100 MB and "Overwrite events as needed" via GPO

HIPAA §164.312 Technical Safeguards

While this tool does not handle ePHI, it is designed to be deployable in HIPAA-regulated environments without introducing compliance gaps.

ℹ️ Scope Note HIPAA Technical Safeguards apply to systems that create, receive, maintain, or transmit electronic Protected Health Information (ePHI). This tool is a system utility — it performs OS-level diagnostics and repairs. It does not interact with any clinical, billing, or patient-facing systems. The controls below demonstrate that deploying this tool does not weaken an organization's HIPAA posture.

Code Signing & Integrity

Code signing ensures the EXE has not been tampered with and identifies the publisher. Required for enterprise deployment and HIPAA integrity controls.

How to Sign the Binary

Option A: Self-Signed Certificate (Testing / Internal)

# 1. Create a self-signed code signing certificate
$cert = New-SelfSignedCertificate `
    -Subject "CN=Acme IT Department" `
    -Type CodeSigningCert `
    -CertStoreLocation "Cert:\CurrentUser\My" `
    -NotAfter (Get-Date).AddYears(3)

# 2. Export the certificate (distribute to clients via GPO)
Export-Certificate -Cert $cert -FilePath "C:\certs\AcmeCodeSign.cer"

# 3. Sign the EXE
Set-AuthenticodeSignature `
    -FilePath ".\publish\DesktopSupportTool.exe" `
    -Certificate $cert `
    -TimestampServer "http://timestamp.digicert.com"

Option B: Commercial Certificate (Production)

Purchase a code signing certificate from DigiCert, Sectigo, or GlobalSign. Then sign with:

signtool sign /a /fd SHA256 /tr http://timestamp.digicert.com /td SHA256 `
    ".\publish\DesktopSupportTool.exe"

Verification

# Verify signature
Get-AuthenticodeSignature ".\publish\DesktopSupportTool.exe" | Format-List *
