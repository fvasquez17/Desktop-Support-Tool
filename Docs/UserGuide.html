🚀 Getting Started

The Desktop Support Tool is a lightweight system tray application that gives IT professionals and end users instant access to system diagnostics, network information, peripheral management, and one-click fixes for common Windows issues.

Launching the App

The tool starts minimized to the system tray (notification area near the clock). It does not open a window on launch.

  • Single-click the tray icon → Opens the main window
  • Right-click the tray icon → Context menu with quick navigation
  • Close (✕) the window → Minimizes back to tray (does not exit)
  • To fully exit → Right-click tray icon → Exit

Window Controls

ButtonAction
— (Minimize)Minimize to taskbar
□ (Maximize)Toggle fullscreen / windowed (respects taskbar)
✕ (Close)Hide to system tray — app keeps running in background

🤖 IT Support Agent

The built-in AI Support Agent is your first stop for troubleshooting. Describe your problem in plain English and the agent will diagnose, scan your system, and offer one-click fixes — all without any internet connection.

How to Use

  1. Click "IT Support Agent" in the sidebar (top item under SUPPORT)
  2. Type your problem in the text box, for example:
    • "Outlook keeps crashing"
    • "I can't print anything"
    • "My computer is really slow"
    • "No internet connection"
  3. The agent scans your system and shows a health bar with live metrics
  4. Click any ▶ fix button to execute a recommended repair
  5. Use quick reply chips (blue pill buttons) for follow-up actions

📊 Dashboard

The home page provides a quick overview of your system's health and key information at a glance.

  • System Summary — Computer name, OS version, CPU, RAM, disk usage
  • Network — Active adapter, IP address, gateway, DNS servers
  • Health Indicator — Green / amber / red dot in the sidebar footer
  • Remote Desktop — Quick launch button to open RDP

🌐 Network

Real-time information about your active network connection. Virtual adapters (Hyper-V, Docker, VMware) are automatically filtered out.

  • Adapter — Name, type (Wi-Fi / Ethernet), MAC address, link speed
  • IPv4 — IP address, subnet mask, default gateway
  • IPv6 — IPv6 address(es) if available
  • DNS — Primary and secondary DNS servers
  • Quick Actions — Flush DNS, Release/Renew IP, Open Network Settings

🖨️ Peripherals

Manage printers, displays, and audio devices from one place.

Printers

  • View all installed printers with online/offline status
  • Set Default — Set any printer as the Windows default
  • Add Printer — Opens the Windows manual add-printer wizard
  • Remove — Uninstall a printer

Display

  • View current resolution and refresh rate
  • Quick access to Windows display settings

Audio

  • View all playback and recording devices
  • Volume slider and mute toggle

🔍 Drivers

Complete system driver inventory with status and problem detection.

  • Driver List — Name, version, date, manufacturer, digital signature
  • Status — Running / Stopped indicator for each driver
  • Problem Flags — Drivers with issues are highlighted with warnings
  • Actions — Update driver, disable driver, open Device Manager

⚙️ Services

View and manage Windows services with start, stop, and restart controls.

  • Service List — Service name, display name, status, startup type
  • Actions — Start, Stop, Restart individual services
  • Search — Type to filter services by name
  • Troubleshoot — Quickly identify stopped services that should be running

📝 Activity Log

Complete audit trail of every action performed in the tool during the current session.

  • Timestamped entries — Date, time, category, and message
  • Severity levels — Info (blue), Warning (amber), Error (red)
  • Search filter — Type to filter log entries in real time
  • Session-based — Logs persist for the current app session

💡 Tips & Tricks

  1. Use the Agent first — describe your issue in plain English before manually browsing the Troubleshooting tab
  2. Right-click the tray icon for quick access to any section without opening the full window
  3. Run "Scan my system" periodically to catch issues before they become problems
  4. Clear Temp + Clear Browsers is a quick way to free up disk space
  5. Restart your PC if uptime exceeds 7 days — many issues resolve after a clean restart
  6. Non-admin fixes (no 🛡️ badge) can be run by any user without IT assistance
  7. Check Services tab if a specific Windows feature isn't working — the required service may be stopped
